# Async Weather Tracker 🌤

A simple weather tracking web app built using **HTML, CSS, and Vanilla JavaScript**.

## Features

* Search weather by city name
* Displays temperature, weather condition, humidity, and wind speed
* Search history for previously searched cities
* Demonstrates **async/await**, **Fetch API**, and **JavaScript event loop logs**

## Tech Used

* HTML5
* CSS3
* JavaScript
* OpenWeather API
